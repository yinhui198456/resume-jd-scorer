## ⏱️ 时间线

| 时间 | 操作 |
|------|------|
| **第零步** | 牛切块焯水；菜心洗净；空心菜洗净；土豆胡萝卜切块；紫菜撕碎；鸡蛋打散；淘米 |
| **0分钟** | 电饭煲煮饭 + 牛腩加土豆胡萝卜+咖喱块炖25分钟 |
| **15分钟** | 白灼菜心：水开→菜心2分钟→淋豉油 |
| **20分钟** | 蒜蓉炒空心菜：大火快炒3分钟 |
| **25分钟** | 紫菜蛋花汤：水开→紫菜虾皮→淋蛋液 |
| **30分钟** | 咖喱牛腩出锅，统一上桌 |

---

## 📝 要点

1. **咖喱牛腩**：牛腩焯水去血沫；咖喱块最后10分钟放，避免糊底
2. **白灼菜心**：水要多，菜心2分钟刚好脆嫩
3. **空心菜**：大火快炒，变色即出锅
