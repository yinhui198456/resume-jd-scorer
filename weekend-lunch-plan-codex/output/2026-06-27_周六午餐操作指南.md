# 2026-06-27 周六午餐操作指南

## 菜单总览

- 【大荤】咖喱牛腩：牛腩500g、土豆2个、胡萝卜1根、洋葱半个、温和咖喱块或咖喱粉、生抽1汤匙、盐约2g、油1汤匙
- 【大荤】蒜蓉粉丝蒸扇贝：扇贝6-8只、粉丝1小把、蒜1头、生抽1汤匙、油1汤匙
- 【素】腐乳炒空心菜：空心菜1把、白腐乳1块、蒜2瓣、油半汤匙
- 【汤】冬瓜丸子汤：冬瓜300g、瘦肉馅250g、姜2片、葱1根、盐约2g、生抽半汤匙、淀粉1小勺
- 【主食】南瓜饭：大米2杯、南瓜250g

预计正餐制作时间：约43分钟。牛腩必须提前压熟，否则当天午餐时间会超。

## 第零步：提前准备

### 牛腩提前压熟

1. 牛腩切3cm块，冷水下锅。
2. 加2片姜，大火煮开后撇沫，继续煮3分钟。
3. 捞出牛腩，用温水冲净。
4. 放入电压锅，加热水没过牛腩。
5. 上汽约8分钟，压20分钟，自然或手动泄压约5分钟。
6. 捞出牛腩，留半碗原汤做咖喱收汁。

### 其他切配

1. 土豆、胡萝卜切小块；洋葱切块。
2. 南瓜去皮切1.5cm小块。
3. 粉丝用温水泡软，剪短。
4. 扇贝刷洗干净；使用冷冻扇贝肉则提前解冻并沥干。
5. 空心菜洗净切段，梗和叶分开放。
6. 冬瓜去皮切薄片。
7. 肉馅加半汤匙生抽、1小勺淀粉、少量葱姜水，顺一个方向搅到发黏。

## 第1道：南瓜饭

1. 大米淘洗2-3遍。
2. 按平时煮饭水量加水，南瓜块铺在米上。
3. 启动电饭煲快煮或标准煮饭。
4. 跳闸后焖5分钟再开盖，轻轻拌匀。

## 第2道：咖喱牛腩

1. 炒锅加1汤匙油，中火下洋葱炒香，约1分钟。
2. 下土豆、胡萝卜翻炒2分钟。
3. 加提前压熟的牛腩和半碗牛腩原汤。
4. 加温水至刚没过食材，中火煮10分钟，至土豆变软。
5. 转小火，放咖喱块或咖喱粉，慢慢搅开。
6. 小火收汁5分钟，期间勤翻动防糊底。
7. 尝味后只少量补盐，装盘。

## 第3道：蒜蓉粉丝蒸扇贝

1. 蒸锅先上汽。
2. 盘底铺粉丝，扇贝放在粉丝上。
3. 蒜末用少量油小火炒到微黄，关火后加1汤匙生抽拌匀。
4. 蒜蓉汁铺到扇贝上。
5. 上汽后蒸6-8分钟。
6. 判断熟度：贝肉完全变乳白，边缘微卷。
7. 出锅后可用少量热油激香蒜蓉。

## 第4道：冬瓜丸子汤

1. 汤锅加水和2片姜，烧到微沸。
2. 用勺子把肉馅团成小丸子，下入锅中。
3. 丸子全部浮起后，再煮3分钟。
4. 下冬瓜片，继续煮8-10分钟。
5. 加约2g盐调味，撒葱花出锅。

## 第5道：腐乳炒空心菜

1. 白腐乳加1汤匙清水压散成汁。
2. 炒锅加半汤匙油，下蒜末炒香。
3. 先下空心菜梗，大火炒30秒。
4. 下菜叶，倒入腐乳汁，大火翻炒1-2分钟。
5. 菜叶刚塌软就出锅，避免出水。

## 上桌顺序

1. 南瓜饭先焖着保温。
2. 咖喱牛腩完成后盖上锅盖保温。
3. 扇贝蒸好立即上桌，口感最好。
4. 冬瓜丸子汤和空心菜最后完成。

## 关键提醒

- 牛腩必须提前压熟；当天只做咖喱收汁。
- 扇贝必须蒸到贝肉乳白、边缘微卷。
- 肉丸浮起后还要再煮3分钟。
- 全餐用油控制在约2.5汤匙以内。
