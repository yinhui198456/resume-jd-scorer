# 2026-06-06 红队审计实战案例

**菜单**：同安封肉、白炒鲜贝、锅塌豆腐、烤蔬菜拼盘、杂粮饭
**标注总时长**：≤45分钟
**审计结论**：❌ FAIL（3项P0否决 + 6项P1-P2问题）

---

## P0-1：时间线矛盾

标注45分钟，实际需52-57分钟。

**漏算项**：
- 高压锅上汽5-10min未计入（标注30min仅压制时间）
- 高压锅泄压3-5min未计入
- 高压锅收汁2-3min未计入
- 烤箱预热5min未计入
- 两道炒锅菜串行+洗锅时间超出预期

**修正后最短时间线**：44分钟（理想最优并行），实际操作中52-57分钟。

## P0-2：焯水串味

猪肉焯水后水质含血沫，直接焯鲜贝会串味且温度不足。

**违背原则**：轻味食材（鲜贝）应在重味食材（猪肉）之后单独用清水处理。

## P0-3：鲜贝杀菌不足

焯水30秒+快炒2分钟，鲜贝中心温度可能仅40-50°C。

**安全标准**：
- 副溶血性弧菌：≥60°C/1min
- 诺如病毒：≥85°C/1min

## 关键教训

1. **高压锅时间必须算完整链路**，不能只算压制时间
2. **不同食材焯水必须换水**，尤其是肉类和海鲜
3. **海鲜焯水时间不能太短**，30秒不足杀菌
4. **所有调料必须量化**，"适量""少许"新手无法执行
5. **烤蔬菜需确认品种适配时间**，根茎类需延长
