## 🐦 盐焗乳鸽（high priority 意向菜）

**来源**：小红书/下厨房高分菜谱验证
**关键步骤**：
1. 乳鸽洗净后**必须沥干水分**（用厨房纸吸干表面和腹腔）
2. 盐焗粉均匀涂抹，**按摩15分钟**入味
3. **冷藏3小时**以上（过夜更佳）
4. 蒸制时**肚皮朝上**，蒸25-30分钟
5. 出锅后**静置3分钟**再斩件（锁汁）

**网友教训**：
- ❌ 不沥干水分 → 盐焗粉化开，味道不均
- ❌ 按摩不足5分钟 → 不入味
- ❌ 蒸完立刻斩 → 汁水流失，肉质干柴

## 🐟 红烧鳜鱼

**来源**：下厨房收藏1万+
**关键步骤**：
1. 鳜鱼洗净，**两面各切3-4刀**（方便入味）
2. 厨房纸**吸干表面水分**（防溅油+不粘锅）
3. 热锅冷油，**鱼皮朝下先煎**，中火2分钟定型后翻面
4. 加葱姜蒜爆香 → 料酒2汤匙 → 生抽2汤匙 → 老抽半汤匙 → 冰糖3粒 → 开水没过鱼身一半
5. 大火烧开转中火**焖8分钟** → 大火收汁2分钟
6. 收汁前2分钟加入**青红椒菱形块**（半个青椒+半个红椒，提色不抢味）
7. 出锅撒葱花

**调料**：料酒2汤匙、生抽2汤匙、老抽半汤匙、冰糖3粒、葱2根、姜3片、蒜4瓣
**网友教训**：
- ❌ 冷水下锅 → 鱼肉收紧变硬，用开水
- ❌ 翻面太早 → 鱼皮破，等定型再翻
- ⚠️ 预留青红椒点缀位

## 🐄 清炖牛肋条

**来源**：美食天下收藏5000+
**关键步骤**：
1. 牛肋条500g切3cm块，**冷水下锅焯水3分钟**（去血水）
2. 撇去浮沫，捞出温水冲洗（不用冷水，肉质收紧）
3. 电压锅中放入：牛肋条、葱2根（打结）、姜5片、料酒2汤匙
4. **上汽后压制30分钟** → 泄压 → 开盖
5. 加白萝卜块300g，继续压10分钟
6. 开盖后加盐1小勺调味，撒枸杞10粒、葱花

**调料**：葱2根、姜5片、料酒2汤匙、盐1小勺、枸杞10粒
**网友教训**：
- ❌ 热水焯肉 → 血水锁在内部，必须冷水下锅
- ❌ 焯水后冷水冲洗 → 肉质收紧，用温水
- ⚠️ 完整链路时间：上汽5-10min + 压制30min + 泄压3-5min + 二压10min + 泄压3min = 约55-65min（需提前启动）

## 🦐 虾仁蒸豆腐

**来源**：下厨房收藏8000+
**关键步骤**：
1. 嫩豆腐1盒（约350g）切1.5cm厚片，铺入盘中
2. 鲜虾仁150g去虾线，用料酒半汤匙+淀粉半小勺腌10分钟
3. 虾仁摆在豆腐上，撒少许盐
4. 水开后上蒸锅，**大火蒸8分钟**
5. 出锅淋蒸鱼豉油1汤匙，撒葱花，浇热油1汤匙

**调料**：料酒半汤匙、淀粉半小勺、蒸鱼豉油1汤匙、葱1根、热油1汤匙
**网友教训**：
- ❌ 虾仁不腌制 → 有腥味
- ❌ 蒸太久 → 豆腐出水变烂，严格8分钟
- ❌ 用老豆腐 → 口感粗糙，必须嫩豆腐

## 🐟 清蒸多宝鱼（历史已做，供参考）

**关键步骤**：
1. 活鱼处理，洗净去鳞去内脏
2. 鱼身两面各切3刀，抹少许盐，放姜片、葱段，**腌制10分钟**
3. 水开后上蒸锅，**大火蒸8分钟**（500g鱼）/ 10分钟（750g鱼）
4. 倒掉蒸出的汤水（腥），取出姜葱
5. 铺上新鲜葱丝、姜丝、红椒丝
6. **蒸鱼豉油2汤匙**沿盘边淋入
7. 烧**2汤匙热油**（冒烟程度）淋在葱姜丝上激香

**网友教训**：
- ❌ 必须活鱼，死鱼清蒸腥味重
- ❌ 蒸出的汤水必须倒掉，否则腥
- ❌ 油温不够 → 葱姜丝不香，油要烧到冒烟

## 🐍 红烧黄鳝 / 响油鳝丝

**红烧黄鳝**：
1. 黄鳝500g让摊主处理去骨，切5cm段
2. **蒜量加倍**（黄鳝配蒜才香）：蒜8-10瓣拍碎
3. 热锅3汤匙油，下蒜爆香 → 黄鳝段翻炒至变色
4. 料酒2汤匙 → 生抽2汤匙 → 老抽半汤匙 → 冰糖3粒 → 开水没过
5. 中火焖12分钟 → 大火收汁 → 青红椒点缀
6. 出锅撒白胡椒粉少许

**响油鳝丝**：
1. 鳝丝300g焯水30秒（去黏液），沥干
2. 葱白2根切丝铺底，鳝丝铺上
3. 浇头：蒜末、葱花、白胡椒粉、少许糖
4. 烧2汤匙热油（冒烟）**浇在蒜末上**激香
5. 沿盘边淋生抽1汤匙

**网友教训**：
- ❌ 蒜不够 → 黄鳝腥味重，必须加倍
- ❌ 鳝丝不焯水 → 黏液影响口感

## 🥬 上汤苋菜（6月TOP1当季）

**关键步骤**：
1. 苋菜400g去老根，洗净沥干
2. 皮蛋1个切丁，咸蛋半个取蛋黄切碎
3. 热锅2汤匙油，下蒜3瓣爆香 → 皮蛋丁翻炒30秒
4. 加入开水400ml，大火煮2分钟出白汤
5. 下苋菜，**大火翻炒1分钟**（苋菜出水快，大火不出水）
6. 加盐半小勺调味，出锅

**网友教训**：
- ❌ 小火炒苋菜 → 出水严重，必须大火快炒
- ⚠️ 六月苋金不换，当季TOP1

## 🐟 煎青花鱼（历史已做，供参考）

**关键步骤**：
1. 青花鱼解冻后**吸干表面水分**（不裹淀粉）
2. 热锅少许油，**鱼皮朝下**先煎
3. 中火3分钟定型 → 翻面煎2分钟
4. 挤柠檬汁，撒少许盐

**网友教训**：
- ❌ 裹淀粉 → 多余，吸干直接煎即可
- ❌ 不吸干 → 溅油+粘锅

## 通用调料量化标准

| 调料 | 直观单位 | 参考量 |
|------|---------|--------|
| 液体调料（酱油/料酒/醋/油） | 汤匙 | 1汤匙≈15ml |
| 盐 | 小勺 | 1小勺≈5g |
| 糖（冰糖） | 粒 | 3粒冰糖≈5g |
| 葱姜蒜 | 瓣/根/片 | 蒜2-4瓣、葱1根（30g）、姜3-5片 |
