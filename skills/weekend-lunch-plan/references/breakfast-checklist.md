# 早餐质量清单

用于 `meal_type=breakfast` 的方案生成和审核。早餐不套用午餐的“大荤/汤/餐厅感”规则，重点是快、稳、营养均衡。

## 结构要求

每套早餐必须包含：

1. 主食：蛋饼、饭团、馄饨、贝果、杂粮粥、蒸点等。
2. 蛋白：鸡蛋、虾仁、鸡胸、奶酪、酸奶、豆制品等。
3. 饮品：牛奶、无糖豆浆、玉米汁、米糊等。
4. 果蔬：小番茄、黄瓜条、蓝莓、香蕉、时令水果等。

## 时间与执行

- 总体制作时间建议控制在 20-30 分钟。
- 人工盯守步骤尽量少，优先使用豆浆机、电蒸锅、电饭煲、空气炸锅。
- 允许前一晚完成浸泡、蒸紫薯、洗切水果等第零步准备。
- 避免早上需要长时间炖煮、复杂摆盘、精准温控。

## 健康与口味

- 低油、低糖、少油烟。
- 不把早餐做成甜点：避免奶油、糖浆、巧克力酱、过量果酱。
- 避免全碳水组合，必须有明确蛋白来源。
- 儿童友好，但不能依靠高糖提高接受度。
- 饮品默认无糖或低糖。

## 食品安全

- 虾仁、鸡胸、肉类必须完全熟透。
- 鸡蛋类必须凝固熟透，不推荐流心蛋给儿童。
- 豆浆必须充分煮熟；使用豆浆机时说明自动完成。
- 冷藏即食食品需确认在保质期内。

## 输出要求

每道早餐项必须标注：

- 类别：主食 / 蛋白 / 饮品 / 果蔬
- 菜名或食品名
- 预计时间
- 库存状态：匹配库存或需采购
- 简短理由
