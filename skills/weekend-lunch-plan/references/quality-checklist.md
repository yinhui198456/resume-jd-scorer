# 菜谱质量校验清单（Quality Checklist）

> 主 agent 自检前置清单。生成初稿方案后、进入 subagent 审核前，按此清单快速扫描。

---

## §1 结构校验
- [ ] 每套方案 = 2大荤 + 1素/小荤 + 1汤 + 1主食
- [ ] 大荤主材必须是荤料（肉/鱼/虾/蟹/贝）
- [ ] 【素】= 0荤料（蛋=荤，皮蛋=蛋=荤）
- [ ] 【小荤】= 含肉/蛋但蔬菜占比 ≥50%

## §2 主材去重
- [ ] 同套方案内主材（归一化后）不重复
- [ ] 主食不参与主材去重
- [ ] 跨方案同一主材出现 ≤2次

## §3 做法去重
- [ ] 同套方案内大荤技法不重复（红烧≠清蒸≠爆炒…）
- [ ] 30天内同「主材+核心做法」不重复

## §4 口味校验
- [ ] 无辣味（辣/麻/藤椒/花椒/红油/剁椒/泡椒/豆瓣酱）
- [ ] 无苦瓜
- [ ] 5-9月无羊肉/狗肉/鹿肉
- [ ] 糖醋/红烧类标注少糖版

## §5 热门验证
- [ ] 优先 popular-dishes.md 标记 ✅ 的菜品
- [ ] 表外菜品有三平台验证依据
- [ ] 无杜撰菜名或「食材A+食材B」机械组合

## §6 当季/创意
- [ ] 每套至少1份当季菜
- [ ] 每套至少1份创意菜（非普通家常菜）
- [ ] 木耳/大蒜/番茄/西兰花/贝贝南瓜不算当季

## §7 库存标注
- [ ] 每道菜标注【匹配库存：xxx】或【无匹配，需采购】

## §8 当季菜判定补充
- 木耳 = 干货/菌菇类，全年可获取，**不算当季**
- 大蒜 = 调味料，**不算当季蔬菜**
- 番茄/西兰花/贝贝南瓜 = 全年供应，**不算当季**
- 6月当季重点：苋菜、空心菜、丝瓜、毛豆、蚕豆（尾声）、豆角、茄子、秋葵、茭白、南瓜藤、黄鳝、蛏子、六月黄

## §9 输出格式纪律

### §9a 方案展示格式
- 飞书消息禁用表格，用纯文本列表
- 格式：`【类别】菜名 — 匹配库存：xxx — 推荐理由`

### §9b 烹饪步骤格式
- 采用「一道一道菜」顺序，每道菜完整做完再做下一道
- **禁止使用 [T+X min] 并行时间线格式**
- 每道菜按「第 X 步：菜名」分段
- 关键节点加粗
- 每道菜末尾标注 `✅ 完成 → 装盘/保温`

## §10 做菜教训速查
- 四季豆/春笋必须焯水
- 菌菇汤每周≤1次
- 清蒸必须活鱼
- 禁止写「清炒时蔬」（太模糊）
- 黄鳝蒜量加倍
- 苋菜大火快炒不出水
- 丝瓜去皮后尽快下锅防氧化发黑
- 空心菜加少许醋保持翠绿
- 茄子盐腌10分钟挤水省油
- 秋葵整根焯水后切段不黏

## §11 鸡蛋频次
- 同一主食材（鸡蛋）在3套方案中出现 ≤2次
- 3套汤都用鸡蛋 → 替换至少1个为无蛋汤

## §12 主食去重
- 主食不可连续3次相同（白米饭/杂粮饭）

## §13 季节性温燥
- 5月-9月（盛夏）排除羊肉、狗肉、鹿肉等温燥食材
- 换清爽海鲜/禽类

## §14 已确认格式清单（用户已确认的偏好）
- 烹饪步骤采用「一道一道菜」顺序（非并行时间线）
- 删除采购清单章节（用户确认不需要）
- 调料用直观单位（汤匙/小勺/瓣/根），不写克数/毫升
- 青红椒点缀位预留在红烧/浓油赤酱类菜品

## §15 汤锅复用纪律
- 方案同时有「汤锅类汤品」+「需要汤锅焯水/白灼的菜」
- 必须写明：汤炖好后转移到砂锅/保温容器，释放汤锅
- 器具分布表必须分开列（电压锅≠汤锅），不可合并写

## §16 特色度/精致度（2026-06-18 新增，用户反馈驱动）

**背景**：Jimmy 明确反馈"方案符合要求但太家常了，没有特色"。家常菜≠没特色，需要在健康底线和可操作性的前提下提升"餐厅感"和精致度。

**判定标准**（候选菜至少满足以下 2 项）：
- **食材组合讲究**：不常见但易买到的搭配（如黄鳝配紫苏、苋菜配上汤、丝瓜配蛤蜊），而非单一主材炒制
- **烹饪手法多样**：蒸+煎、炖+烤、热油激香等组合，避免单一炒/煮
- **菜名有记忆点**：不用"红烧XX""清炒XX"泛名，用"紫苏焖黄鳝""蒜蓉蒸丝瓜""响油鳝丝"等具体技法名
- **视觉亮点**：摆盘有层次、颜色对比鲜明（青红椒点缀/粉丝铺底/浇头激香），适合拍照或让孩子有新鲜感
- **仪式感**：上桌时有动作（热油浇蒜末激香、揭盖蒸汽、汤汁沸腾声）

**禁止**：
- 3 套方案全部是"清炒 XX""红烧 XX""XX 炒蛋"式家常菜
- 纯素菜用"清炒时蔬"敷衍
- 菜名与做法脱节（如"XX 炒 XX"实际只是简单翻炒）

**每套方案必须至少 1 道菜满足"有特色"标准**（创意菜定义见 SKILL.md，此处是其补充：不仅要求技法新颖，还要求视觉/仪式感/菜名记忆点）。
