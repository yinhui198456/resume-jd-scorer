##  使用方式

### 新方式（推荐）

```bash
# 启动混合模式会话（错题+到期复习）
python tools/quiz_cli.py start mixed

# 启动错题复习会话
python tools/quiz_cli.py start wrong

# 获取题目
python tools/quiz_cli.py get wrong 5
python tools/quiz_cli.py get new 3 M14_推理部署 M15_成本优化

# 提交答案
python tools/quiz_cli.py submit Q-M14_推理部署09 C

# 查看统计
python tools/quiz_cli.py stats
```
